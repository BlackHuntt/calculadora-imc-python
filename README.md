# Calculadora de IMC em Python

Calculadora de Índice de Massa Corporal com validação de dados e classificação da OMS.

### Funcionalidades:
- Validação de entrada: não aceita letras ou valores absurdos
- Cálculo preciso do IMC com 2 casas decimais
- Classificação completa segundo tabela da OMS
- Loop para múltiplos cálculos
- Interface amigável no terminal

### Como executar:
```bash
python calculadora_imc.py
